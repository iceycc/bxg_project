### 准备

1. 初始化项目

```bash
vue create todomvc-vue-vuex
cd todomvc-vue-vuex
npm run serve
```

2. 下载模板

```bash
git clone https://github.com/tastejs/todomvc-app-template.git
cd todomvc-app-template
npm i
```

3. 将模板整合到项目中
